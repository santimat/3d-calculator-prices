# Calculadora para obtener precios de productos impresos en 3D

## Descripción

Esta calculadora permite a los usuarios obtener precios aproximados para productos impresos en 3D, teniendo en cuenta factores como el material utilizado, el tiempo de impresión y la complejidad del diseño. Es una herramienta útil para diseñadores, fabricantes y cualquier persona interesada en la impresión 3D.

## Características

- Configuración por default via archivo de configuración.
- Historial de cálculos realizados.
- Interfaz de usuario amigable y fácil de usar.
- Suma de productos agregados a la impresión para obtener un precio total.
- Registro de productos vendidos con un detalle y su precio.
- Vista de Ganancias, Costos y Ventas.
