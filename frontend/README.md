# Calculadora Profesional de Precios para Impresión 3D

## Descripción General

Esta aplicación permite calcular de manera precisa y rápida el precio de productos impresos en 3D, considerando variables relevantes como el tipo de material, tiempo de impresión, complejidad del diseño y productos adicionales. Es una solución pensada tanto para profesionales como para aficionados que desean optimizar costos, gestionar ventas y obtener reportes detallados sobre sus operaciones de impresión 3D.

## Características Principales

- Fácil configuración inicial mediante archivo de parámetros personalizables.
- Historial completo de cálculos de precios previamente realizados.
- Interfaz de usuario intuitiva y amigable para facilitar el flujo de trabajo.
- Suma automática de productos y servicios asociados a cada impresión.
- Registro de ventas detallado, incluyendo desglose de productos y precios.
- Panel de control para visualizar ganancias netas, costos totales y estadísticas de ventas.

## Historial y Reportes

- El sistema almacena automáticamente cada cálculo realizado y brinda la posibilidad de revisarlo posteriormente.
- Puedes consultar reportes de ganancias, costos y ventas desde la sección de estadísticas.
